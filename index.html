Sobre el proyecto

Este proyecto forma parte de una práctica académica sobre control de versiones. El objetivo es modificar estos archivos, registrar los cambios con Git y publicarlos en un repositorio remoto.

  • 1 Clona o inicializa el repositorio.
  • 2 Realiza cambios en el código.
  • 3 Haz git add y git commit.
  • 4 Sube tus cambios con git push.
